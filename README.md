# RiggedRPS
Rigged Rock Paper Scissors

## Purpose
This project was created to practice Python fundamentals and probability concepts.

## Description
The program plays Rock, Paper, Scissors against the user. Unlike a normal game, the computer adjusts its choices based on the player's previous moves to increase its chances of winning.

## Concepts Practiced
- Variables
- Functions
- Lists
- Dictionaries
- Loops
- Random module
- Basic probability

## Why I Built It
I built this project as part of my journey learning Python and backend development. The goal was to move beyond beginner exercises and apply several concepts in a single program.
